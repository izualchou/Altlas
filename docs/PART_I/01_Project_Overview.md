# Chapter 01 - Project Overview

## Purpose
Atlas is an AI-driven Tasker XML engineering platform.

## Vision
Transform natural language requirements into deterministic, production-ready Tasker XML through metadata-driven generation and validation.

## Scope
- Task XML
- Profile XML
- Project XML
- Tasker metadata
- Java Function
- Runtime architecture
- XML validation

## Core Principles
- XML-first
- Metadata-first
- Validation-first
- Incremental deployment
- Deterministic generation
